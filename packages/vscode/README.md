# Context Doctor for VS Code

Analyze and optimize LLM context windows from VS Code.

Features:

- Status bar token count for active file.
- Dashboard with tokens, context use, cost, largest segment, waste, and one-click fixes.
- Waste highlights with hover suggestions.
- Sidebar breakdown by model, compression style, segment, and waste item.
- Commands for analyze, optimize, compare with clipboard, model select, style select.

Settings:

- `contextDoctor.defaultModel`
- `contextDoctor.compressionStyle`
- `contextDoctor.warningThreshold`
- `contextDoctor.criticalThreshold`
- `contextDoctor.autoAnalyze`
